# Input Menu

The input menu is where you can find various input generators, either provided by Avogadro or downloaded in the [Extensions Menu](menus-extensions-menu).

By default, Avogadro comes packaged with several input generators, which you should be able to see in the input menu as long as you have properly configured Python.

```{figure} /_static/menus/input-menu/input_dropdown.png
:alt: Image of the input menu
:align: center
:height: 400px
:class: dark-light
```

APBS
: The [Adaptive Poisson-Boltzmann Solver](https://www.poissonboltzmann.org/) software.

CP2K
: The [CP2K Open-Source Molecular Dynamics Software](https://www.cp2k.org/).

GAMESS
: The [General Atomic and Molecular Electronic Structure System (GAMESS)](https://www.msg.chem.iastate.edu/gamess/) (US version).

LAMMPS
: The [Large-scale Atomic/Molecular Massively Parallel Simulator (LAMMPS)](https://www.lammps.org/#gsc.tab=0).

OpenMM Script
: The [Open Molecular Mechanics (OpenMM)](https://openmm.org/) software.

Dalton
: The [Dalton](https://gitlab.com/dalton/dalton) program.

GAMESS-UK
: The [UK version of GAMESS](https://doi.org/10.1080/00268970512331340592).

Gaussian
: The [Gaussian](https://gaussian.com/) software.

MOLPRO
: The [Molpro](https://www.molpro.net/) software.

MOPAC
: The [Molecular Orbital PACkage](https://openmopac.net/).

NWChem
: The [NWChem](https://nwchemgit.github.io/index.html) program.

ORCA
: The [ORCA](https://www.faccts.de/orca/) program.

Psi
: The [Psi4](https://psicode.org/) program.

PySCF
: The [Python Self-Consistent Field (PySCF)](https://pyscf.org/) program.

Q-Chem
: The [Q-Chem](https://www.q-chem.com/) program.

: The [TeraChem](https://mtzweb.stanford.edu/software) software.
