Release Notes v1.96.0#

June 2, 2022

Changes#

We are pleased to announce the latest release of Avogadro, including piles of bug fixes and feature enhancements, including contributions from @aerkiaga as part of Google Summer of Code. Thanks to many for suggestions, bug reports, and discussions.

Draft developer documentation can be found at https://two.avogadro.cc

As part of this release, we have an official AppImage build for Linux, as well as a code-signed release for macOS.

If you have comments or suggestions, please join us at https://discuss.avogadro.cc and as always, help and contributions in many forms are very welcome, particularly if you‚Äôd like to help with user documentation or website developement.
