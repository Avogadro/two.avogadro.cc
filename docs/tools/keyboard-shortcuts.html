Keyboard#

Three equivalent sets of directional keys are supported: Arrow keys, WASD, and HJKL.

  • ↑ = W = K (referred to as up below)

  • ↓ = S = J (down)

  • ← = A = H (left)

  • → = D = L (right)

Shortcut

Action

Directional keys

Rotate (z-axis)

Ctrl + directional keys

Pan (x/y)

Hold Alt together with any keyboard navigation action to increase the step size.

Warning

Ctrl+W, Ctrl+A, and Ctrl+S are mapped to other functions and cannot be used for panning. Use the arrow keys or HJKL instead.

Tip

Template Tool shortcuts are case-sensitive. For example, c6 selects hexyl while C6 selects cyclohexane.

Note

Multi-character shortcuts have a 2-second timeout. The buffer clears automatically after 5 characters if no match is found.
